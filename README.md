# 槟城野生漫游 - Penang Wild Roam

## 项目结构

基于 Next.js 15 + Tailwind CSS 重构的槟城旅游落地页。

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建静态站点
npm run build
```

## 配色系统

| 名称 | 色值 | 用途 |
|------|------|------|
| sand | #F5E6D3 | 沙色背景 |
| ocean | #2B6A8E | 海洋蓝 |
| coral | #E07B54 | 珊瑚橙/CTA |
| forest | #3D6B4F | 森林绿 |
| night | #1A1A2E | 深夜蓝/深色背景 |
| cream | #FDF8F2 | 奶油色背景 |

## 页面区块

1. **Hero** - 全屏主视觉
2. **Stats Strip** - 数据统计条
3. **About** - 关于罗拉
4. **Photo Strip** - 四图拼接
5. **Experiences** - 体验亮点
6. **Itinerary** - 样版一天
7. **Why Us** - 为什么选择我们
8. **Testimonials** - 用户评价
9. **Lola** - 关于罗拉详情
10. **Products** - 旅程产品
11. **Data Insights** - 数据洞察
12. **CTA Bar** - 行动召唤
13. **Contact** - 联系表单
14. **Footer** - 页脚

## 图片素材

图片放置在 `public/images/` 目录下。当前使用占位色块，替换时取消注释 Image 组件。
